This is the setup phase.
Click on a block in the game grid to select it.
When a block is selected you can press a certain key on the keyboard to place an item there.
- A - press this to place an asteroid
- M - press this to place a mine
- R - press this to place a robotic spaceship
- U - press this to place the users spaceship
- Delete - press this to empty the current block
